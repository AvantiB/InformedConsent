# InformedConsent

Code for the informed-consent data-use and sharing project.

## Repository structure

```text
meaning_preservation/   # MVP classifier experiments for round-trip meaning preservation
meta_model/             # reserved for later data-driven meta-model construction
```

The current implementation focuses on the meaning-preservation classifier used as a proxy evaluator for LLM round-trip experiments.

See [`meaning_preservation/README.md`](meaning_preservation/README.md) for setup and usage.
